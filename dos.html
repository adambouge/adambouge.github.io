Addition

It adds 2 numbers together

cls
@echo off
echo "The first number is %1"
echo "The second number is %2"
set /a result=%1 + %2
echo "The addition is: %result%"

Simple IP scanner

It finds active IPs on the local network

@echo off
for /L %%i in (1,1,255) do (
echo 192.168.0.%%i
ping -n 1 192.168.0.%%i
if %errorlevel% equ 0 echo 192.168.0.%%i is online
)  
pause
